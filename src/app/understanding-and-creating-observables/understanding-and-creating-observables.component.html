Understanding and creating Observables

An Observable is a representation of any set of values over any amount of time.

Observers

An Observer is observing the data the Observable produces. They implement the observer interface, which has the methods next() , error() and complete()

The methods corresponds to the 3 types of messages an Observable can produce: They push the next value in the data stream to their observer with next, if the get an error they call the error message and when done producing values they call complete.

Subscribers

Objects that want to recieve values from an Observable are passed to a method named subscribe()
