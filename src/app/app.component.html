Intro - Reactive Programming Basics

RXJS is a popular library, often used with frameworks like Angular. It takes it ques from functional programming.

It is a library for building asynchronous applications with observable sequences (a stream of data).

With vanilla javascript, the code for handling a function call, a promise and a click event is different. But RXJS provides a nice abstraction over all of these techniques, with a single API.
